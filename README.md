# Prosjektoppgave-Uke46
En webside egnet for å sende meldinger.

Jeg skal lage en webside lik Facebook som lar deg sende meldinger/videoer. En miks mellom Facebook/twitter  på det sosiale aspektet og ordinær sms/imessage. Fokuset er på melding-aspektet. Jeg skal antagelig bruke Html, css, Python, server(kanskje flask/mariadb). Jeg skal demonstrere kompetanse i 3 fag ved å vise hvordan websiden funker, (vise, fortelle, informere).
